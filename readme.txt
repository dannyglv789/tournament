Tournament_Results_Project 
	- tournament.sql
		contains our table schema and views
	- tournament.py 
		Define our functions to complete various queries and display results
	- tournament_test.py
		tests against tournament.py to verify functionality

INSTRUCTIONS (assuming python is installed): 
In order to run please 
1. configure and install Vagrant https://docs.vagrantup.com/v2/installation/index.html
2. if not already installed please install bleach https://bleach.readthedocs.org/en/latest/
3. place Tournament_Results_Project files in your shared folder
4. Run tournament_test.py uncommenting bottom functions to perform tasks
